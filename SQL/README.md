# SQL Scripts
This folder contains the SQL scripts required for database and table creation, as well as sample data insertion.

FindMyMeal Database
This project contains the SQL script for the FindMyMeal database. The database supports user accounts, recipes, and saved recipe functionality.

Tables
User: Stores user information.
Recipe: Stores recipe details.
UserRecipe: Links users with their saved recipes.
How to Use
Run the SQL script in a MySQL database.
The database will create tables and insert sample data.
Use queries to interact with the data.
