# Alternative Matrix
This folder contains the alternative matrix document for Deliverable 4. It evaluates various solutions against key criteria to choose the best option for implementation.

## Criteria Used
1. Cost
2. Scalability
3. Flexibility
4. Maintenance
5. Security
6. Time to Implement
7. Integration
