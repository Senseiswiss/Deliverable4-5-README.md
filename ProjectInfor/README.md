## Project Information

### Team Name
**FindMyMeal**

### Project Overview
FindMyMeal is a web-based meal planning and recipe-finding application that leverages APIs to provide recipes, pricing, and nutritional information. Users can search by cuisine, view recipe history, and save favorites.

### Technologies Implemented
Our project is built using the following technologies:

1. **Frontend:**
   - **HTML/CSS/JavaScript**: For designing and implementing the user interface.
   - **React.js**: To create dynamic and interactive frontend components.

2. **Backend:**
   - **Node.js**: For server-side scripting and handling API requests.
   - **Express.js**: As the web application framework for backend development.

3. **Database:**
   - **MySQL**: To store and manage user, recipe, and related data.
   - **Docker**: For creating a containerized database environment.

4. **APIs:**
   - **Spoonacular API**: To fetch recipes, nutritional information, and pricing.

5. **System Architecture:**
   - **Cloud-Based Architecture**: Hosting and deployment on cloud platforms for scalability and reliability.
   - **AWS**: For cloud hosting, database backups, and monitoring.

6. **Tools:**
   - **GitHub**: For version control and project management.
   - **Lucidchart/Visio**: For creating diagrams like ERDs and architecture designs.
   - **Visual Studio Code**: As the primary code editor for development.

7. **Testing Frameworks:**
   - **Postman**: For API testing.
   - **Jest**: For unit testing of backend functionalities.
   - **Selenium**: For automated frontend testing.

8. **Communication & Collaboration:**
   - **Discord**: For team communication.
   - **Google Docs**: For collaborative documentation.

9. **Other:**
   - **Bootstrap**: For styling and responsive design.
   - **REST APIs**: For seamless communication between frontend and backend.
