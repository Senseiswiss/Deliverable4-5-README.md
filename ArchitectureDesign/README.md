# Architecture Design

This folder contains the architecture design components for the Deliverable 4 project. It explores various architecture models for the system and provides the final selected design for implementation.

## Architecture Models
The following architectural options were evaluated for the project:
1. **Server-Based Architecture**
2. **Client-Based Architecture**
3. **Thin-Client Server Architecture**
4. **Thick-Client Server Architecture**
5. **Cloud Computing Architecture**

## Files in this Folder
- **server-based_architecture.png**: Diagram illustrating the server-based architecture.
- **client-based_architecture.png**: Diagram illustrating the client-based architecture.
- **thin-client_server_architecture.png**: Diagram illustrating the thin-client server architecture.
- **thick-client_server_architecture.png**: Diagram illustrating the thick-client server architecture.
- **cloud_computing_architecture.png**: Diagram illustrating the cloud computing architecture.
- **findmymeal.com_architecture.png**: The finalized architecture design for the system.

## How to Use
- Review the diagrams to understand each architecture's structure.
- Refer to the `RequirementMatrix.png` file for the chosen critera for final architecture. 
- Refer to the `findmymeal.com_architecture.png` file for the final design selected for implementation.

## Key Decisions
The final architecture was chosen based on the following criteria:
- Scalability
- Ease of Maintenance
- Cost Efficiency
- Performance
- Low latency
- Ease of Deployment
- Encryption/Authentication
